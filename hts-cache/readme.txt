O que há nesta pasta?

Esta pasta (hts-cache) foi gerada pelo WINHTTRACK 3.49-2+htswf+htsjava
e é usado para atualizar este site.
(A estrutura do site HTML é armazenada aqui para permitir atualizações rápidas)

Não exclua esta pasta, a menos que você não queira atualizar o espelho no futuro !!
(Você pode excluir com segurança os arquivos antigos.zip e antigo.lst, no entanto)

Nota: o arquivo hts-log.txt e a pasta HTS-cache podem conter informações confidenciais,
como a autenticação de nome de usuário/senha para sites espelhados neste projeto
Não compartilhe esses arquivos/pastas se quiser que essas informações permaneçam privadas
